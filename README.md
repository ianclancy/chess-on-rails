# Chess on Rails

## Technologies
* Ruby on Rails back-end with PostgreSQL database
* React.js front-end

## Setup
To get set up, clone this repository and run:
```
bundle
rake db:create
rake db:migrate
rails s
```

In a separate terminal window, run:
```
npm install
npm start
```

In your browser, navigate to:
```
http://localhost:3000/
```
